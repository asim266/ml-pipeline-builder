# Privacy Policy

**ML Pipeline Builder Plugin**
Last updated: March 9, 2025

## Data Collection

This plugin does **not** collect, store, transmit, or process any personal data, usage data, telemetry, analytics, or any other information from users. It operates entirely within the user's local environment.

## What This Plugin Does

This plugin provides instructional guidance, code examples, Python scripts, and best practices for building machine learning pipelines. It does not:

- Connect to any external servers or APIs
- Send or receive data over the network
- Store any user information locally or remotely
- Track usage, behavior, or analytics
- Access credentials, tokens, or API keys
- Collect any form of personally identifiable information (PII)

## Third-Party Services

This plugin does not integrate with or send data to any third-party services.

## Security

The bundled scripts (`quick-eda.py`, `baseline-comparison.py`, and `data-leakage-check.py`) run entirely within the user's local environment. They read local files and process data locally only. No data from these scripts is transmitted externally.
